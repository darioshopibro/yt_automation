# Infographic Types Catalog

> Research compiled from web sources + YouTube tutorials for tech explainer videos

---

## 1. Table (BASIC)
**Svrha:** Structured data, feature lists, comparisons, specs

**Vizualni elementi:**
- Rows and columns
- Header row (bold/colored)
- Alternating row colors (optional)
- Cell highlighting
- Icons in cells

**Primer upotrebe:**
- Feature comparison tables
- Pricing tiers
- Specs/requirements
- Any structured data

**Animation style:** Row-by-row reveal, cell highlight

**Status:** HIGH PRIORITY - najčešći vizual

---

## 2. VS Layout (META-WRAPPER)
**Svrha:** Side-by-side comparison of ANY two visuals

**Vizualni elementi:**
- Split screen (50/50 or custom ratio)
- "VS" badge in middle
- Left panel: any visual component
- Right panel: any visual component
- Labels per side

**Primer upotrebe:**
- Table vs Table
- Chart vs Chart
- Node diagram vs Node diagram
- Timeline vs Timeline
- Bilo šta vs Bilo šta

**Animation style:** Split reveal, alternating focus, winner highlight

**Status:** HIGH PRIORITY - reusable wrapper

---

## 3. Code Block / Snippet (BASIC)
**Svrha:** Display code with syntax highlighting

**Vizualni elementi:**
- Dark background (terminal style)
- Syntax highlighting (colors per language)
- Line numbers (optional)
- File name header
- Typing animation
- Highlight specific lines

**Primer upotrebe:**
- Code examples
- "Here's how you do it" moments
- API calls
- Config files
- Any code demonstration

**Animation style:** Typing animation, line-by-line reveal, highlight flash

**Status:** HIGH PRIORITY - essential za tech content

---

## 4. Terminal / Console (BASIC)
**Svrha:** Command line interface display

**Vizualni elementi:**
- Black/dark background
- Green/white text
- Command prompt ($, >, etc.)
- Typing animation for commands
- Output appearing after command
- Cursor blinking

**Primer upotrebe:**
- "Run this command"
- Installation steps
- CLI tool demos
- Build/compile output
- Error messages

**Animation style:** Command typing, output reveal, cursor blink

**Status:** HIGH PRIORITY - svaki tutorial ima ovo

---

## 5. List / Bullet Points (BASIC)
**Svrha:** Simple lists, key points, features

**Vizualni elementi:**
- Bullet points or numbers
- Icons per item (optional)
- Checkmarks
- Indentation for sub-items
- Highlight current item

**Primer upotrebe:**
- "3 reasons why..."
- Feature lists
- Requirements
- Key takeaways
- Any enumeration

**Animation style:** Item-by-item reveal, checkmark animation

**Status:** HIGH PRIORITY - super basic, koristi se stalno

---

## 6. Logo Grid / Tech Stack (BASIC)
**Svrha:** Display multiple logos/technologies

**Vizualni elementi:**
- Grid layout (2x2, 3x3, etc.)
- Logo images
- Labels under logos
- Category grouping
- Connection lines (optional)

**Primer upotrebe:**
- "Built with..." sections
- Tech stack overview
- Integrations
- Supported platforms
- Tool comparisons

**Animation style:** Logo pop-in, grid fill, glow on hover/focus

**Status:** HIGH PRIORITY - "tech stack" u svakom drugom videu

---

## 7. Node/Flow Diagram (EXISTING)
**Svrha:** Architecture, connections, tech stack, data flow

**Vizualni elementi:**
- Nodes (boxes/circles)
- Connecting lines/arrows
- Labels/icons
- Optional: animated data flow along lines

**Primer upotrebe:**
- "How the system works" explanations
- API architecture
- ML pipeline visualization
- Tech stack overview

**Status:** Already implemented as `node-canvas` template

---

## 8. Timeline
**Svrha:** Chronological progression, history, evolution

**Vizualni elementi:**
- Horizontal or vertical line
- Milestone points/markers
- Dates/labels
- Icons per milestone
- Progress animation

**Primer upotrebe:**
- "History of AI" segments
- Version/release history
- "First... then... finally" narratives
- Evolution of technology

**Animation style:** Sequential reveal, point-by-point

---

## 9. Data Visualization / Charts
**Svrha:** Statistics, trends, comparisons, metrics

**Vizualni elementi:**
- Bar charts (vertical/horizontal)
- Line charts
- Pie/donut charts
- Racing bar charts (time-series)
- Animated number counters

**Primer upotrebe:**
- "X% of companies use..."
- Performance benchmarks
- Growth statistics
- Market share comparisons

**Animation style:** Bars growing, lines drawing, numbers counting up

---

## 10. Comparison / Versus
**Svrha:** Side-by-side comparison, pros/cons, A vs B

**Vizualni elementi:**
- Split screen (left/right)
- Comparison table
- Check/X marks
- Balance scale visual
- Highlighting differences

**Primer upotrebe:**
- "Python vs JavaScript"
- "Old way vs new way"
- Feature comparison
- Pros and cons lists

**Animation style:** Alternating reveals, highlighting winner

---

## 11. Process Steps / How-To
**Svrha:** Step-by-step instructions, workflow, sequences

**Vizualni elementi:**
- Numbered steps (1, 2, 3...)
- Arrows between steps
- Icons per step
- Progress indicator
- Checkmarks on completion

**Primer upotrebe:**
- "How to set up X"
- Installation guides
- Workflow explanations
- Tutorial segments

**Animation style:** Sequential reveal with completion marks

---

## 12. Map Animation
**Svrha:** Geographic context, regional data, location-based info

**Vizualni elementi:**
- World/country map
- Highlighted regions
- Zoom in/out
- Location pins/markers
- Data overlays on regions

**Primer upotrebe:**
- "Available in these countries"
- Data center locations
- Regional statistics
- Expansion narratives

**Animation style:** Zoom, region highlight, pin drops

---

## 13. Statistics / Metrics Display
**Svrha:** Key numbers, KPIs, impressive stats

**Vizualni elementi:**
- Large animated numbers
- Percentage bars/circles
- Icons with metrics
- Grid of stats
- Emphasis effects (glow, pulse)

**Primer upotrebe:**
- "10 million users"
- "99.9% uptime"
- Speed/performance metrics
- Feature highlights

**Animation style:** Count-up animation, progress fill

---

## 14. Hierarchy / Tree Structure
**Svrha:** Organization, nested relationships, categorization

**Vizualni elementi:**
- Tree diagram (top-down or left-right)
- Parent-child connections
- Expandable nodes
- Category groups
- Organizational structure

**Primer upotrebe:**
- File system structure
- API endpoint hierarchy
- Class inheritance diagrams
- Organization charts

**Animation style:** Branch expansion, level-by-level reveal

---

## 15. Isometric / 2.5D
**Svrha:** Spatial relationships, product visualization, depth

**Vizualni elementi:**
- Angled perspective (pseudo-3D)
- Stacked layers
- Building blocks
- Room/environment scenes
- Components assembly

**Primer upotrebe:**
- Product architecture
- System components
- "Under the hood" explanations
- Tech stack layers

**Animation style:** Assembly, rotation, layer reveals

---

## 16. Educational Diagram / Labeled Illustration
**Svrha:** Technical concepts, anatomy, component breakdown

**Vizualni elementi:**
- Central illustration/object
- Label lines/arrows
- Callout boxes
- Highlight zones
- Zoom on details

**Primer upotrebe:**
- "Parts of a neural network"
- Hardware component breakdown
- Technical concept visualization
- Feature annotation

**Animation style:** Labels appearing, zoom + highlight

---

## 17. Kinetic Typography
**Svrha:** Text emphasis, quotes, key messages

**Vizualni elementi:**
- Moving/animated text
- Size/color changes
- Word-by-word reveal
- Text transformations
- Sync with voiceover

**Primer upotrebe:**
- Important quotes/statements
- Key takeaways
- Emphasis moments
- Title sequences

**Animation style:** Word reveal, scale emphasis, bounce

---

## 18. Icon Grid / Feature Matrix
**Svrha:** Feature lists, capabilities, multi-point overview

**Vizualni elementi:**
- Grid of icons
- Labels under each
- Category grouping
- Animated icon transitions
- Check/highlight states

**Primer upotrebe:**
- "Features include..."
- Capability overview
- Tool/tech stack icons
- Multi-point summaries

**Animation style:** Grid fill-in, sequential icon reveal

---

## 19. Split Screen / Dual Panel
**Svrha:** Before/after, cause/effect, dual narratives

**Vizualni elementi:**
- Two panels side by side
- Divider line
- Contrasting content
- Synchronized animations
- Labels per side

**Primer upotrebe:**
- "Before AI vs After AI"
- Problem vs Solution
- User A vs User B
- Input vs Output

**Animation style:** Split reveal, parallel animations

---

## 20. Animated Counter / Ticker
**Svrha:** Real-time feel, dynamic stats, urgency

**Vizualni elementi:**
- Spinning/counting numbers
- Dollar/currency displays
- Progress trackers
- Live-style counters

**Primer upotrebe:**
- Revenue growth
- User acquisition
- Real-time metrics
- "Every second..." statements

**Animation style:** Number scroll, continuous update feel

## Summary - Priority for Tech Explainer Videos

| Priority | Type | Use Case |
|----------|------|----------|
| **HIGH** | **1. Table** | Structured data, specs, feature lists |
| **HIGH** | **2. VS Layout** | Any visual vs any visual (wrapper) |
| **HIGH** | **3. Code Block** | Code examples, syntax highlight |
| **HIGH** | **4. Terminal** | Commands, CLI demos |
| **HIGH** | **5. List/Bullets** | Key points, features, enumeration |
| **HIGH** | **6. Logo Grid** | Tech stack, integrations |
| **HIGH** | 7. Node/Flow Diagram | Architecture, how things connect |
| **HIGH** | 8. Timeline | History, evolution, progression |
| **HIGH** | 9. Data Visualization | Stats, benchmarks, comparisons |
| **HIGH** | 10. Comparison | A vs B, pros/cons |
| MEDIUM | 11. Process Steps | How-to, tutorials |
| MEDIUM | 13. Statistics Display | Key metrics, impressive numbers |
| MEDIUM | 14. Hierarchy | System structure, organization |
| LOW | 12. Map Animation | Geographic context (niche) |
| LOW | 15. Isometric | Product viz (more design-heavy) |
| LOW | 17. Kinetic Typography | Emphasis moments |
