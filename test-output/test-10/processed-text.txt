DevOps is a set of practices to build, test, and release your code in small frequent steps. Now anytime we commit code to the master branch in this repo, it will run this workflow. If any of the steps fail, the bad software won't be delivered to our customers, and we'll automatically know there's an issue that needs to be addressed. At the end of the day, C I C D offers two main benefits. It helps you automate things that would otherwise have to be done manually by developers. That will increase your velocity. But it also detects small problems early before they can grow into major disasters, and that results in higher code quality.
